# 📄 Resume Analyzer

🚀 Smart Resume Analyzer built using Python & Streamlit.

This app analyzes resumes, extracts skills (even with spelling mistakes), and suggests the best job roles based on skill matching.

---

## 🔥 Features

- 📄 Upload Resume (PDF, DOCX, TXT)
- 🧠 Automatic Skill Extraction
- ✨ Handles Spelling Mistakes (Fuzzy Matching)
- 🎯 Best Job Role Suggestion
- 🏆 Top 3 Role Recommendations
- 📊 Match Score Calculation
- 📌 Missing Skills Detection

---

## 🛠️ Tech Stack

- Python
- Streamlit
- RapidFuzz (Fuzzy Matching)
- PyPDF2 (PDF Processing)
- python-docx (DOCX Processing)
- Regex (Text Cleaning)

---

## 📂 How to Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
